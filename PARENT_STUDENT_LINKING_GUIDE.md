# Parent-Student Linking Guide

## How to Link Students with Parents

### Method 1: Add Parent First (Recommended)

1. **Create Parent Account First**
   - Click "Add User" button
   - Select "Parent" card
   - Fill in parent information:
     - Username, Email, Password
     - First Name, Last Name
     - Phone Number (important for identification)
     - Relationship (Father/Mother/Guardian)
   - Click "Add Parent"
   - Click "Yes" when asked to add student

2. **Add Student with Parent Link**
   - Fill in student information
   - In "Parent/Guardian" dropdown, select the parent you just created
   - The dropdown shows: "Name - Phone Number" format
   - Click "Add Student"

### Method 2: Add Student First, Then Parent

1. **Add Student Without Parent**
   - Click "Add User" → "Student"
   - Fill in all student information
   - Leave "Parent/Guardian" dropdown as "No Parent Account (Optional)"
   - Click "Add Student"

2. **Create Parent Later**
   - Click "Add User" → "Parent"
   - Fill in parent information
   - Click "Add Parent"

3. **Link Them Together** (Future Feature)
   - Edit the student record
   - Select the parent from dropdown
   - Save changes

### Method 3: Quick Parent Creation from Student Form

1. **Start Adding Student**
   - Click "Add User" → "Student"
   - Start filling student information

2. **Notice Parent Doesn't Exist?**
   - See the yellow info box: "Need to add a parent first?"
   - Click "Add Parent First" button
   - Confirm the dialog

3. **Add Parent**
   - Parent form opens
   - Fill in parent information
   - Click "Add Parent"
   - Confirm to add student

4. **Complete Student Registration**
   - Student form reopens
   - Fill in student information
   - Select parent from dropdown (now available)
   - Click "Add Student"

## Understanding the Fields

### Why Blood Group?

**Purpose**: Medical emergency information

- Required by most schools for safety
- Used when student needs medical attention
- Helps medical staff respond quickly
- **Field is OPTIONAL** - you can leave it blank

**When to fill it:**

- If you have the student's blood group from medical records
- If parents provided this information
- For boarding schools or residential programs

**When to skip:**

- Information not available
- Parents haven't shared
- Student is too young for blood donation records

### Parent/Guardian Field

**Purpose**: Link student to their parent account

- Parent can view student's attendance
- Receive notifications about their child
- Access student reports
- Track student activities

**Options:**

- Select existing parent from dropdown
- Shows: "FirstName LastName - Phone"
- Leave blank if parent account doesn't exist yet
- Can be updated later

## Parent Dropdown Options

```
Jane Smith - 555-5678

```

### If Dropdown is Empty:

- No parent accounts created yet
- Create parent account first
- Then return to add student

## Tips for Better Parent-Student Linking

1. **Use Parent's Phone Number**
   - Helps identify correct parent
   - Shown in dropdown for easy selection
   - Used for emergency contact

2. **Create Parents First**
   - Easier workflow
   - Can link multiple students to same parent
   - Dropdown already populated

3. **One Parent, Multiple Students**
   - Create parent once
   - Add first student, link parent
   - Add second student, link same parent
   - Siblings automatically connected

4. **Update Later**
   - Don't have parent info? Skip it
   - Add parent account when available
   - Edit student to add link (future feature)

## Common Questions

**Q: Can I add student without parent?**
A: Yes! Parent field is optional. Add parent later.

**Q: Can multiple students link to one parent?**
A: Yes! Perfect for siblings.

**Q: Can one student have multiple parents?**
A: Currently one parent per student. Add primary guardian.

**Q: What if I don't know the blood group?**
A: Leave it blank. It's optional and can be added later.

**Q: Can I change the parent link later?**
A: Yes, through edit function (coming soon).

## Security Note

- Parents can only see their linked students
- Students see their own information only
- Admin can see and manage all accounts
- Teachers can view students in their classes
